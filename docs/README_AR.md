# 🎵 زايلو (Xylo) — منصة دعم صُنّاع المحتوى

> منصة عربية متكاملة تربط الكُتّاب والمبدعين بالقرّاء، وتُتيح للقرّاء دعم كُتّابهم المفضّلين من خلال نظام هدايا افتراضية يُحوَّل إلى أرباح حقيقية.

---

## 📌 نظرة عامة على المشروع

**زايلو** هو تطبيق ويب مبني بتقنيات حديثة يهدف إلى تمكين صُنّاع المحتوى العربي من نشر مقالاتهم وكسب الأرباح عبر نظام عملات افتراضية مزدوج. يستطيع القرّاء شراء عملة **MALCOIN** بأموال حقيقية وإرسال هدايا للكُتّاب، بينما يحصل الكُتّاب على أرباحهم بعملة **QUSCOIN** التي يمكن تحويلها إلى أموال حقيقية.

---

## ✨ مميزات المشروع

### ✍️ محرر مقالات متقدم
- محرر نصوص غني مبني على **Tiptap** يدعم التنسيق الكامل (خط عريض، مائل، تسطير، روابط، محاذاة النص، صور).
- إمكانية إنشاء ونشر المقالات بسهولة.

### 📱 اكتشاف المقالات
- تصفّح المقالات والبحث فيها وتصفيتها.
- عرض بيانات تفصيلية لكل مقال (المشاهدات، الإعجابات، التعليقات، الهدايا).

### 💳 نظام العملات المزدوج
| العملة | الوصف | سعر الصرف |
|--------|-------|------------|
| **MALCOIN** (عملة الشحن) | يشتريها المستخدم بأموال حقيقية لإرسال الهدايا | ١ دولار = ٥٠٠ MALCOIN |
| **QUSCOIN** (عملة الأرباح) | يحصل عليها الكاتب من الهدايا المستلمة | ٨٠٪ من قيمة الهدية للكاتب، ٢٠٪ عمولة المنصة |

### 🎁 نظام الهدايا
- يستطيع القرّاء إرسال هدايا افتراضية متنوعة لدعم الكُتّاب المفضّلين.
- تُحوَّل قيمة الهدايا تلقائيًا إلى أرباح للكاتب.

### 👤 إدارة الملفات الشخصية
- نظام تسجيل دخول وإنشاء حسابات آمن.
- ملفات شخصية تتضمن الاسم المعروض، الصورة الرمزية، والنبذة التعريفية.
- نظام توثيق الحسابات.

### 💰 إدارة المحفظة
- تتبع الرصيد لعملتي MALCOIN و QUSCOIN.
- باقات شحن متعددة مع مكافآت ترويجية.
- سجل كامل للمعاملات المالية.
- نظام سحب الأرباح وتحويلها إلى أموال حقيقية.

### 💸 نظام سحب الأرباح
- دعم بوابات دفع متعددة:
  - Stripe
  - PayPal
  - Moyasar
  - STC Pay
  - Payoneer
  - Skrill

### 📊 لوحة التحليلات
- عرض إحصائيات أداء المقالات.
- متابعة المشاهدات والتفاعلات والأرباح.

### 🔗 نظام الإحالات
- مكافأة تسجيل بقيمة ٥٠ دولارًا بعملة MALCOIN.
- عمولة ٢٪ على نشاط المستخدمين المُحالين.

### 🔔 نظام الإشعارات
- إشعارات فورية للهدايا والتعليقات والإحالات والتحديثات العامة.

### 🛡️ لوحة تحكم المسؤول
- إدارة المستخدمين وطلبات السحب وباقات الشحن.
- عرض إحصائيات المنصة الشاملة.

## 🚀 خطوات تشغيل المشروع

### المتطلبات الأساسية

- [Node.js](https://nodejs.org/) (الإصدار ١٨ أو أحدث)
- npm أو yarn
- حساب على [Supabase](https://supabase.com/)

### ١. استنساخ المستودع

```bash
git clone https://github.com/yayass3r/TickXyol.git
cd TickXyol
```

### ٢. الانتقال إلى مجلد التطبيق

```bash
cd xylo-app
```

### ٣. تثبيت الحزم والاعتماديات

```bash
npm install
```

### ٤. إعداد متغيرات البيئة

أنشئ ملف `.env.local` داخل مجلد `xylo-app` وأضف المتغيرات التالية:

```env
NEXT_PUBLIC_SUPABASE_URL=رابط_مشروع_Supabase_الخاص_بك
NEXT_PUBLIC_SUPABASE_ANON_KEY=مفتاح_Supabase_العام
JWT_SECRET=مفتاح_سري_لتوقيع_JWT
```

### ٥. إعداد قاعدة البيانات

- افتح لوحة تحكم Supabase الخاصة بمشروعك.
- انتقل إلى **SQL Editor**.
- انسخ محتوى ملف `supabase/schema.sql` وشغّله لإنشاء الجداول والبنية المطلوبة.

### ٦. تشغيل خادم التطوير

```bash
npm run dev
```

ثم افتح المتصفح على العنوان: [http://localhost:3000](http://localhost:3000)

## 🗄️ هيكل قاعدة البيانات

| الجدول | الوصف |
|--------|-------|
| `users` | حسابات المستخدمين وأدوارهم (مستخدم، كاتب، مشرف، مسؤول) |
| `wallets` | أرصدة MALCOIN و QUSCOIN لكل مستخدم |
| `recharge_packages` | باقات شحن MALCOIN المتاحة |
| `transactions` | جميع حركات المحفظة (شحن، هدايا، سحب، إحالات) |
| `gifts` | أنواع الهدايا المتاحة وتكلفتها |
| `gift_transactions` | سجل الهدايا المرسلة بين المستخدمين |
| `articles` | المقالات المنشورة وبياناتها |
| `comments` | التعليقات على المقالات |
| `withdrawal_requests` | طلبات سحب أرباح الكُتّاب |
| `referrals` | تتبع الإحالات والمكافآت |
| `notifications` | إشعارات المستخدمين |

## 👥 أدوار المستخدمين

| الدور | الصلاحيات |
|-------|----------|
| **مستخدم (USER)** | تصفح المقالات، إرسال الهدايا، شحن المحفظة |
| **كاتب (CREATOR)** | جميع صلاحيات المستخدم + نشر المقالات وسحب الأرباح |
| **مشرف (MODERATOR)** | مراجعة المحتوى والإشراف عليه |
| **مسؤول (ADMIN)** | إدارة كاملة للمنصة عبر لوحة التحكم |
